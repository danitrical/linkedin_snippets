# 🪨 LinkedIn Infinity Saga

> *"I am inevitable."* — And so is consistent content creation.

Welcome to the **Infinity Saga** — a living collection of my LinkedIn posts, code snippets, diagrams, and everything I share with the tech community. Think of each post as an **Infinity Stone**: alone they're powerful, together they tell a bigger story.

## 🗺️ What's Inside

This repository is organized by topic. Each folder contains the post text, any accompanying code, and supporting visuals.

```
linkedin-infinity-saga/
│
├── ai-and-agents/          # Agentic workflows, LLMs, RAG pipelines
├── cloud-and-devops/       # AWS, GCP, Docker, Kubernetes, CI/CD
├── fullstack/              # React, Next.js, Node, NestJS, FastAPI
├── career-and-growth/      # Lessons, reflections, job search insights
├── assets/                 # Shared images, diagrams, banners
└── README.md
```

Each post folder follows this structure:

```
ai-and-agents/
└── 2026-04-08-rag-pipeline-deep-dive/
    ├── POST.md              # The LinkedIn post text
    ├── code/                # Any code snippets or scripts
    │   └── rag_example.py
    └── images/              # Diagrams or visuals used
        └── architecture.png
```

## 🚀 Why This Exists

I believe in **building in public**. Every post here represents something I learned, built, or debugged. This repo serves three purposes:

1. **Accountability** — Writing forces clarity of thought
2. **Shareability** — Code in a post deserves a proper home with syntax highlighting and context
3. **Portfolio** — A timeline of growth that goes beyond a static resume
